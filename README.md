## 📚 投資用語集 - ユーザー向け

**アプリで使われる投資・分析用語をわかりやすく解説**

### 📈 **基本的な株価用語**

#### 💰 **株価の基本**

株価とは、その会社の株式1株あたりの値段のことです。例えば3,000円と表示されている場合、その会社の株を1株買うのに3,000円必要ということを意味します。銘柄コードは会社を識別するための番号や記号で、7203.T（トヨタ）やAAPL（Apple）のように表記されます。

| 用語 | 意味 | 例 |
|------|------|-----|
| **株価** | その会社の株式1株あたりの値段 | 3,000円 |
| **銘柄コード** | 会社を識別する番号や記号 | 7203.T（トヨタ）、AAPL（Apple） |
| **終値** | その日の最後に取引された値段 | 当日の最終価格 |
| **始値** | その日の最初に取引された値段 | 当日の開始価格 |
| **高値** | その日の最も高い値段 | 当日の最高価格 |
| **安値** | その日の最も安い値段 | 当日の最安価格 |
| **出来高** | その日に取引された株数 | 1,000万株 |

終値は、その日の取引時間終了時点での最後の取引価格を表し、通常は「当日の最終価格」として重要視されます。始値は取引開始時の最初の価格で、前日の終値と比較することで市場の開始時の反応を知ることができます。高値と安値は、その日の価格変動の幅を示し、市場のボラティリティ（変動性）を理解する上で重要な指標です。出来高は取引された株数の総計で、市場の活発さや投資家の関心の高さを表します。

#### 📊 **トレンド（方向性）**

| 用語 | 意味 | どんな時？ |
|------|------|----------|
| **上昇トレンド** | 株価が上がり続けている状況 | 買い需要が強い、業績好調 |
| **下降トレンド** | 株価が下がり続けている状況 | 売り圧力が強い、悪材料 |
| **レンジ相場** | 株価が一定の範囲で上下している状況 | 様子見、材料待ち |
| **調整** | 大きく上がった後の一時的な下落 | 利益確定売り |
| **反発** | 下がった後の一時的な上昇 | 割安感からの買い |

上昇トレンドは、投資家の買い意欲が強く、企業の業績が好調である場合によく見られます。企業の売上増加や新製品の成功などが要因となることが多く、市場全体が楽観的な雰囲気の時に発生しやすい現象です。

下降トレンドは、何らかの悪材料や市場全体の不安により、売り圧力が買い圧力を上回る状況です。企業の業績悪化や経済情勢の悪化などが要因となり、投資家が株式を売却したがる心理状態を表します。

レンジ相場は、明確な方向性が見えない状況で、投資家が様子見を決め込んでいる状態です。重要な発表や市場の材料を待っている状況で、一定の価格帯で株価が行ったり来たりを繰り返します。

調整は、株価が大きく上昇した後に起こる一時的な下落で、これは利益を確定したい投資家による売りが原因です。基本的なトレンドは変わらず、健全な市場動向の一部として理解されます。

反発は、株価が下落した後に起こる一時的な上昇で、「安くなったから買おう」という投資家の心理により発生します。割安感からの買いが入ることで、一時的に価格が回復する現象です。

### 📊 **テクニカル分析用語**

#### 📈 **移動平均線**

| 用語 | 意味 | 見方 |
|------|------|------|
| **移動平均線** | 過去一定期間の平均株価を線で表示 | トレンドの方向を見る |
| **短期移動平均** | 直近20日程度の平均 | 短期的な動きを表す |
| **長期移動平均** | 直近50日程度の平均 | 長期的な動きを表す |
| **ゴールデンクロス** | 短期線が長期線を上に突破 | 🟢 買いサイン |
| **デッドクロス** | 短期線が長期線を下に突破 | 🔴 売りサイン |

**📚 初心者向け解説**: 移動平均線は「株価の平均点」のようなものです。学校のテストで考えると、最近のテストの平均点が上がっていれば成績が向上している、下がっていれば成績が悪化していると判断できるのと同じ原理です。株価でも、移動平均線が上向きなら株価が上昇傾向、下向きなら下降傾向と判断できます。

短期移動平均線は最近の株価動向をより敏感に反映し、市場の直近の変化を捉えやすい特徴があります。長期移動平均線は長期的なトレンドを示し、一時的な価格変動に惑わされることなく、大きな流れを把握できます。

ゴールデンクロスは、短期的な上昇の勢いが長期的なトレンドを上回った状況を示し、多くの投資家が買いサインとして注目します。デッドクロスは逆の現象で、短期的な下落の勢いが強まったことを示す売りサインとして認識されます。

#### 🌡️ **RSI（相対力指数）**

| 用語 | 意味 | 数値の見方 |
|------|------|----------|
| **RSI** | 買われすぎ・売られすぎを0-100%で表示 | 株価の「熱」を測る体温計 |
| **買われすぎ** | RSI 70%以上 | 🔴 そろそろ下がるかも |
| **売られすぎ** | RSI 30%以下 | 🟢 そろそろ上がるかも |
| **中立** | RSI 30-70% | ⚪ 普通の状態 |

**📚 初心者向け解説**: RSIは株価の「興奮度」を測る体温計のようなものです。人間の体温で考えると、熱が出すぎている時（70%以上）は休憩が必要で、体温が低すぎる時（30%以下）は元気になる可能性があります。株価でも同様に、RSIが高すぎる時は価格が下がる可能性があり、低すぎる時は上がる可能性があると考えられます。

買われすぎの状態（RSI 70%以上）では、多くの投資家がすでに株を購入しており、新たに買う人が少なくなる可能性があります。そのため、利益確定の売りが出やすく、価格が下がる傾向があります。

売られすぎの状態（RSI 30%以下）では、多くの投資家が株を売却しており、割安感から新たに買う投資家が現れやすくなります。そのため、価格が反発上昇する可能性が高まります。

中立状態（RSI 30-70%）では、買いと売りのバランスが取れており、明確な方向性が見えない状況です。他の指標と組み合わせて総合的に判断することが重要です。

#### 📊 **ボリンジャーバンド**

ボリンジャーバンドは、株価の「いつもの範囲」を3本の線で表示する分析手法です。中央の線は移動平均線で、上下の線はその移動平均線から統計学的に計算された標準偏差を示します。これにより、株価が通常動く範囲を視覚的に把握できます。

| 用語 | 意味 | 見方 |
|------|------|------|
| **ボリンジャーバンド** | 株価の「いつもの範囲」を表示 | 3本の線で価格の動く範囲を示す |
| **上部バンド** | 通常の上限価格ライン | 価格が届くと「高すぎる」可能性 |
| **下部バンド** | 通常の下限価格ライン | 価格が届くと「安すぎる」可能性 |
| **バンド幅拡大** | 3本の線の幅が広がる | ボラティリティ（変動）が大きい |
| **バンド幅縮小** | 3本の線の幅が狭まる | ボラティリティ（変動）が小さい |

**📚 初心者向け解説**: ボリンジャーバンドは、その人の「普段の行動範囲」を示すようなものです。例えば、普段家と学校を往復している人が、突然遠くに行くと「いつもと違う」と感じるのと同じで、株価が上部バンドや下部バンドに触れると、「いつもより高すぎる」「いつもより安すぎる」と判断できます。

上部バンドに株価が接近または到達した場合、統計学的には「買われすぎ」の状態と考えられ、価格が下がる可能性が高まります。逆に下部バンドに接近または到達した場合は「売られすぎ」の状態で、価格が上がる可能性が高まります。

バンド幅の変化も重要な情報を提供します。バンド幅が拡大している時は市場のボラティリティが高く、大きな価格変動が起こりやすい状況です。バンド幅が縮小している時は市場が落ち着いており、大きな変動が起こりにくい状況を示します。

このように、ボリンジャーバンドは価格の相対的な位置とその時の市場の変動性を同時に把握できる、非常に有用な分析ツールです。他のテクニカル指標と組み合わせることで、より精度の高い分析が可能になります。

## 🏠 ローカル実行（中・上級者）

ローカル環境でアプリケーションを実行する場合、まずGitHubリポジトリからプロジェクトをクローンします。

依存関係のインストールには「pip install -r requirements.txt」を実行し、必要なPythonライブラリをすべてインストールします。すべての依存関係がインストールされたら、「streamlit run app.py」でアプリケーションを起動します。ブラウザで http://localhost:8501 にアクセスすることで、ローカル環境でTradeSim v2.1を使用できます。

### ⚙️ 環境変数設定（上級者向け）

より高度な機能を利用したい場合は、プロジェクトルートディレクトリに .env ファイルを作成し、API認証情報を設定します。

ファイル内には 
JQUANTS_EMAIL=your-email@example.com、
JQUANTS_PASSWORD=your-password、
ALPHA_VANTAGE_API_KEY=your-api-key 

の形式で記述します。これにより、JQuantsとAlpha VantageのAPIを自動的に利用でき、より高度な検索機能と分析機能を使用できます。

## 🔐 セキュリティとプライバシー

### API認証情報の取り扱い

API認証情報については、最高レベルのセキュリティ対策を実施しています。JQuants APIの認証情報（メールアドレスとパスワード）は、セッション中のみメモリ内に保持され、セッション終了と同時に自動的にクリアされます。パスワード入力欄には「type="password"」設定により、画面上でのマスク表示を徹底しています。

Alpha Vantage APIキーについても、同様にセッション中のみの保持となっており、サーバー側での永続的な保存は一切行いません。これにより、ユーザーの認証情報が第三者に漏洩するリスクを最小限に抑えています。

### プライバシー保護

プライバシー保護については、認証情報がセッション中のみ保持され、ブラウザを閉じた瞬間に自動的に削除される設計となっています。サーバー側での認証情報保存は一切行わず、ユーザーごとに独立したセッション管理により、他のユーザーとの情報混在を防いでいます。

すべての通信はHTTPS暗号化通信により保護され、データの送受信過程での盗聴や改ざんを防止しています。これらの多層的なセキュリティ対策により、安心してアプリケーションを利用できる環境を提供しています。

## 🐛 トラブルシューティング

### よくある問題と解決方法

#### 1. 🇯🇵 JQuants認証エラー

JQuants API認証に失敗した場合、まずメールアドレスとパスワードが正確に入力されているかを確認してください。特に、大文字小文字の違いや余分なスペースが入力されていないかを注意深くチェックします。JQuantsアカウントが有効であることを確認し、必要に応じてJQuantsの公式サイトでログインテストを行ってください。

一時的なサービス停止やメンテナンスの可能性もあるため、しばらく時間を置いてから再試行することも効果的です。それでも問題が解決しない場合は、JQuantsの公式サポートに問い合わせることをおすすめします。

#### 2. 🌍 Alpha Vantage制限エラー

Alpha Vantage APIの制限に達した場合、無料プランでは1日あたりの制限があるため、24時間後に再試行してください。より頻繁に利用したい場合は、Alpha Vantageの有料プランへのアップグレードを検討するか、検索頻度を調整して効率的に利用することをおすすめします。

API制限の管理により、重要な分析に集中して利用することで、無料プランでも効果的な学習が可能です。どうしても急ぎで分析が必要な場合は、JQuants APIや銘柄コード直接入力による分析を活用してください。

#### 3. 銘柄コード入力エラー

銘柄コードの入力エラーが発生した場合、まず銘柄コードの形式が正しいかを確認してください。日本株の場合は「7203.T」のように証券コード+「.T」、米国株の場合は「AAPL」のようにティッカーシンボルのみとなります。

Yahoo Financeや証券会社のサイトで正確な銘柄コードを再確認し、コピー&ペーストを活用することで入力ミスを防げます。大文字小文字の区別や、余分なスペースが入力されていないかも重要なチェックポイントです。

#### 4. チャート表示エラー

チャート表示に問題がある場合、ブラウザのキャッシュクリアやページの再読み込みを試してください。古いキャッシュが残っていると、正常に表示されない場合があります。

異なるブラウザ（Chrome、Firefox、Safari等）での動作確認も効果的です。インターネット接続が安定していることを確認し、必要に応じてネットワーク環境を改善してください。
