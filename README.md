# 📱 TradeSim - 株価分析学習アプリ

教育・学習目的の株価分析とバックテストシミュレーションアプリケーション。初心者でも理解しやすいインターフェースで、テクニカル分析とポートフォリオ管理を学べます。

## 🌐 ライブデモ

**[📱 今すぐ使ってみる！](https://tradesim-9djzkadr39vp8jt2dxcgud.streamlit.app/)**

> 🔰 ブラウザで即座に利用可能！インストール不要でお試しいただけます。

## ✨ 主な機能

### 📊 インテリジェント株価分析
- **多言語検索**: 日本語・英語での会社名検索（「トヨタ」「Apple」「任天堂」等）
- **リアルタイム検証**: 銘柄コード入力時の自動検証とEnterキー対応
- **テクニカル分析**: 移動平均線、RSI、ボリンジャーバンド、MACDによる多角的分析
- **AI判定**: 複数指標を統合した買い・売り・中立シグナル生成

### 💰 リアリスティック投資シミュレーション
- **バックテスト機能**: 過去データを使った投資戦略の検証
- **リスク管理**: 損切り・利益確定ルールの実装
- **手数料考慮**: 実際の取引コストを含む現実的なシミュレーション
- **パフォーマンス分析**: シャープレシオ、最大ドローダウン等の指標

### 💼 ポートフォリオ管理
- **ウォッチリスト**: 注目銘柄の保存と追跡
- **投資成績追跡**: 投資金額、現在価値、損益の自動計算
- **分散度分析**: ポートフォリオの分散度スコア算出

### 🎓 教育的インターフェース
- **段階的学習**: 初心者から上級者まで対応
- **分かりやすい説明**: 専門用語を平易に解説
- **視覚的チャート**: インタラクティブなチャート表示
- **モバイル対応**: レスポンシブデザインでスマートフォン最適化

4. **学習**: 
   - 詳細な解説と判断根拠

## 📈 機能詳細

### テクニカル分析指標

| 指標 | 説明 | 使用目的 |
|------|------|----------|
| **移動平均線** | 短期・長期の価格平均 | トレンド判定 |
| **RSI** | 相対力指数（0-100） | 買われすぎ・売られすぎ判定 |
| **ボリンジャーバンド** | 価格の標準的な変動範囲 | 価格位置の相対評価 |
| **MACD** | 移動平均収束拡散 | モメンタム分析 |
| **出来高** | 取引量分析 | 市場参加度評価 |

### バックテスト機能

#### リスク管理設定
- **初期資金**: 10万円 〜 1000万円
- **リスク許容率**: 0.5% 〜 5.0%
- **損切り率**: 1.0% 〜 20.0%
- **利益確定率**: 2.0% 〜 50.0%
- **取引手数料**: 0.0% 〜 1.0%

#### パフォーマンス指標
- **総リターン**: 投資成果の絶対値・パーセンテージ
- **シャープレシオ**: リスク調整後リターン
- **最大ドローダウン**: 最大下落幅
- **勝率**: 利益を上げた取引の割合

### プリセット設定

| モード | 対象者 | 設定内容 |
|-------|--------|----------|
| **🔰 初心者向け** | 投資初心者 | 超安全設定（リスク1%、損切り3%） |
| **⚖️ バランス型** | 一般投資家 | 標準設定（リスク2%、損切り5%） |
| **🚀 積極型** | 上級者 | ハイリスク設定（リスク5%、損切り8%） |
