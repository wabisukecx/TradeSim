# 📱 TradeSim - 株価分析学習アプリ

教育・学習目的の株価分析とバックテストシミュレーションアプリケーション。初心者から上級者まで、テクニカル分析とポートフォリオ管理を直感的に学べるWebアプリです。

## 🌐 ライブデモ

**[📱 今すぐ使ってみる！](https://tradesim-9djzkadr39vp8jt2dxcgud.streamlit.app/)**

> 🔰 ブラウザで即座に利用可能！インストール不要でお試しいただけます。

## ✨ 主な機能

- **📊 多角的テクニカル分析**: 移動平均線、RSI、ボリンジャーバンド、MACDによる総合判定
- **🔍 スマート銘柄検索**: 日本語・英語での会社名検索（「トヨタ」「Apple」等）
- **💰 リアリスティック投資シミュレーション**: 過去データを使った戦略検証
- **💼 ポートフォリオ管理**: 注目銘柄の追跡と投資成績分析
- **🎓 教育的インターフェース**: 専門用語を平易に解説、段階的学習支援
- **📱 モバイル対応**: レスポンシブデザインでスマートフォン最適化

## 📊 使用方法

### 基本的な流れ

1. **銘柄選択**
- 会社名検索（例：「トヨタ」「Apple」）
- 銘柄コード直接入力（例：「AAPL」「7203.T」）
1. **分析実行**
- 「🚀 分析開始」ボタンクリック
- または銘柄コード入力後にEnterキー
1. **結果確認**
- AI判定とテクニカル分析結果
- インタラクティブチャート表示
- バックテスト結果とパフォーマンス指標
1. **ポートフォリオ管理**
- 気になる銘柄をウォッチリストに追加
- 投資成績の追跡と分析

### テクニカル分析指標

|指標           |説明          |使用目的         |
|-------------|------------|-------------|
|**移動平均線**    |短期・長期の価格平均  |トレンド判定       |
|**RSI**      |相対力指数（0-100）|買われすぎ・売られすぎ判定|
|**ボリンジャーバンド**|価格の標準的な変動範囲 |価格位置の相対評価    |
|**MACD**     |移動平均収束拡散    |モメンタム分析      |
|**出来高**      |取引量分析       |市場参加度評価      |

### 機能別設定

#### プリセット設定

- **🔰 初心者向け**: 超安全設定（リスク1%、損切り3%）
- **⚖️ バランス型**: 標準設定（リスク2%、損切り5%）
- **🚀 積極型**: ハイリスク設定（リスク5%、損切り8%）

#### バックテスト設定

- **初期資金**: 10万円 〜 1000万円
- **リスク許容率**: 0.5% 〜 5.0%
- **損切り率**: 1.0% 〜 20.0%
- **利益確定率**: 2.0% 〜 50.0%
- **取引手数料**: 0.0% 〜 1.0%
